# Personal Industry Intelligence

This repository is the working home for a personal industry intelligence knowledge base focused on ETRM and CTRM research, starting with Endur, Findur, Openlink, and related material.

The immediate goal is to create a practical, local-first system for organizing documents, notes, extracted text, metadata, and future search tooling. Over time, this project may grow into a more structured ingestion, indexing, and retrieval workflow for personal research and AI-assisted analysis.

## Purpose

This project is intended to support:

- Personal industry research and reference
- Structured document intake and classification
- Metadata-driven organization rather than deep folder taxonomies
- Future local search, indexing, and AI-assisted workflows
- Clear separation between private reference material and content safe for broader reuse

## Knowledge Base Location

The primary knowledge base directory should live inside this repository so the core project structure, scripts, and documentation remain together.

The expected long-term direction is that the main knowledge base content, indexing assets, and project-owned notes live under this project.

At the same time, this project may also reference documents and folders outside the repository. External material may be used as part of:

- Source-Docs
- My-Notes

This allows the project to keep a clear project home while still supporting curated references that live elsewhere on the local machine or in other managed folders.
